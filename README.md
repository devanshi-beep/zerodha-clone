# Zerodha Clone (MERN Stack)

A full-stack learning clone of Zerodha's trading platform: a landing page,
authentication, and a dashboard with a Watchlist, Holdings, Positions,
Orders and Funds — built with MongoDB, Express, React (Vite) and Node.

This is an educational project with **simulated/demo prices**, not a real
brokerage or live market data feed.

## Folder structure

```
zerodha-clone/
  backend/     Express + MongoDB API (auth, holdings, positions, orders, funds)
  frontend/    React (Vite) app — landing page + dashboard
```

See each folder's own instructions below to run locally, and the
**Deployment** section at the end to put it online.
